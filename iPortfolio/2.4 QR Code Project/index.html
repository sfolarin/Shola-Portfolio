QR Code Generator

This project allows you to generate a QR code from a URL entered by the user. The QR code image is saved as qr_img.png and the URL is saved in URL.txt.

How to Run the Project

Follow the instructions below to run the project on your local machine:

1. Download or clone the project repository:
   https://github.com/yourusername/qr-code-generator

2. Navigate to the project directory:
   cd qr-code-generator

3. Install the dependencies:
   npm install

4. Run the project:
   npm start

Dependencies

This project uses the following npm packages:

  • inquirer: For user input
  • qr-image: To generate QR codes
  • fs: Native Node.js module for file system operations
